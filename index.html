How it works

◈

ERC-20 + NFT hybrid

Each whole bPEG token automatically carries a unique pixel art NFT. Buy 1 bPEG → get 1 NFT. Sell → NFT burns. Same mechanic as uPEG on Ethereum.

⬡

100% on-chain SVG

Every image is generated and stored directly on Base. No IPFS, no external servers. As long as Base exists, your bPEG exists.

★

4 rarity levels

Common (60%), Uncommon (25%), Rare (12%) and Ultra Rare (3%). Each level unlocks more ornaments, accessories and visual effects.

⬛

Pixel art blocks

Each bPEG is an isometric pixel art block — a tribute to Base and the blockchain itself. 8 colors, 5 ornaments, 5 accessories, 4 eye types.
